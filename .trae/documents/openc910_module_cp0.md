# OpenC910 一级模块文档：协处理器0 (CP0)

## 1. 模块概述

协处理器0 (Coprocessor 0, CP0) 是 OpenC910 处理器的系统控制核心，负责管理和控制处理器的各种系统功能。CP0 包含控制和状态寄存器 (CSR)，处理异常、中断、特权级切换等系统级操作。

### 1.1 主要功能

- CSR 寄存器管理
- 异常处理
- 中断控制
- 特权级管理
- 性能计数器控制
- 调试支持

## 6. 异常处理机制

### 6.1 异常类型

| 编码 | 异常类型 |
|-----|---------|
| 0 | 指令地址错误 |
| 1 | 指令访问错误 |
| 2 | 非法指令 |
| 3 | 断点 |
| 4 | Load地址错误 |
| 5 | Load访问错误 |
| 6 | Store地址错误 |
| 7 | Store访问错误 |
| 8 | 用户态ECALL |
| 9 | 超级用户态ECALL |
| 11 | 机器态ECALL |
| 12 | 指令页错误 |
| 13 | Load页错误 |
| 15 | Store页错误 |

### 6.2 异常处理流程

```
1. 异常发生
2. 保存当前PC到mepc
3. 设置mcause
4. 设置mtval
5. 设置mstatus.MPP
6. 切换到机器模式
7. 跳转到mtvec指向的地址
```

## 7. 中断处理

### 7.1 中断源

| 中断 | 编码 | 来源 |
|-----|------|------|
| USI | 0 | 用户软件中断 |
| SSI | 1 | 超级用户软件中断 |
| MSI | 3 | 机器软件中断 |
| UTI | 4 | 用户定时器中断 |
| STI | 5 | 超级用户定时器中断 |
| MTI | 7 | 机器定时器中断 |
| UEI | 8 | 用户外部中断 |
| SEI | 9 | 超级用户外部中断 |
| MEI | 11 | 机器外部中断 |

### 7.2 中断优先级

```
MEI > MSI > MTI > SEI > SSI > STI > UEI > USI > UTI
```

## 9. 设计要点

1. **完整CSR支持**: 支持RISC-V标准CSR
2. **嵌套异常**: 支持异常嵌套处理
3. **中断委托**: 支持中断委托到低特权级
4. **PMP集成**: 完整的物理内存保护
