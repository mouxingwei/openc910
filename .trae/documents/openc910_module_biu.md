# OpenC910 一级模块文档：总线接口单元 (BIU)

## 1. 模块概述

总线接口单元 (Bus Interface Unit, BIU) 是 OpenC910 处理器与外部总线系统的接口，负责处理所有的总线事务。BIU 实现 AXI4 协议，支持多核一致性扩展 (ACE)，提供高带宽、低延迟的总线访问。

### 1.1 主要功能

- AXI4 总线协议实现
- 总线请求仲裁
- ACE 一致性协议支持
- 地址翻译支持
- 总线错误处理

## 5. AXI4 协议

### 5.1 信号列表

| 信号 | 宽度 | 描述 |
|-----|------|------|
| AWADDR | 40 | 写地址 |
| AWLEN | 8 | 突发长度 |
| AWSIZE | 3 | 传输大小 |
| AWBURST | 2 | 突发类型 |
| AWCACHE | 4 | 缓存属性 |
| AWPROT | 3 | 保护属性 |
| ARADDR | 40 | 读地址 |
| ARLEN | 8 | 突发长度 |
| ARSIZE | 3 | 传输大小 |
| ARBURST | 2 | 突发类型 |
| WDATA | 128 | 写数据 |
| WSTRB | 16 | 写选通 |
| RDATA | 128 | 读数据 |
| RRESP | 2 | 读响应 |
| BRESP | 2 | 写响应 |

### 5.2 突发类型

| 类型 | 编码 | 描述 |
|-----|------|------|
| FIXED | 00 | 固定地址 |
| INCR | 01 | 递增地址 |
| WRAP | 10 | 回绕地址 |

### 5.3 响应编码

| 响应 | 编码 | 描述 |
|-----|------|------|
| OKAY | 00 | 正常 |
| EXOKAY | 01 | 独占访问成功 |
| SLVERR | 10 | 从设备错误 |
| DECERR | 11 | 解码错误 |

## 6. ACE 一致性扩展

### 6.1 ACE 通道

| 通道 | 描述 |
|-----|------|
| AC | Snoop 地址通道 |
| CR | Snoop 响应通道 |
| CD | Snoop 数据通道 |

### 6.2 Snoop 操作

| 操作 | 描述 |
|-----|------|
| ReadShared | 读取共享副本 |
| ReadClean | 读取干净副本 |
| ReadNotSharedDirty | 读取非共享脏副本 |
| ReadShared | 读取共享副本 |
| CleanInvalid | 清除并无效 |
| MakeInvalid | 使无效 |

## 7. 总线配置

| 参数 | 值 |
|-----|-----|
| 数据宽度 | 128位 |
| 地址宽度 | 40位 |
| ID宽度 | 4位 |
| 突发长度 | 最大16 |

## 8. 设计要点

1. **AXI4 兼容**: 完全符合 AXI4 协议规范
2. **ACE 支持**: 支持多核缓存一致性
3. **高带宽**: 128位数据通路
4. **低延迟**: 优化的仲裁策略
