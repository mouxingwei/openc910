# ct_iu_top 模块设计文档

## 1. 模块概述

### 1.1 基本信息

| 属性 | 值 |
|------|-----|
| 模块名称 | ct_iu_top |
| 文件路径 | iu/rtl/ct_iu_top.v |
| 功能描述 | 整数执行单元顶层模块 |
| 设计特点 | 支持双ALU、乘除法、分支执行 |

### 1.2 功能描述

ct_iu_top 是 OpenC910 处理器的整数执行单元，负责执行整数运算和分支指令。主要功能包括：

- **整数运算**：加减、逻辑、移位等操作
- **乘除运算**：整数乘法和除法
- **分支执行**：分支条件判断和跳转
- **多发射支持**：支持双发射执行

### 1.3 设计特点

- 双ALU设计支持双发射
- 流水线乘法器
- 非阻塞除法器
- 支持分支预测验证
