# OpenC910 一级模块文档：向量浮点单元 (VFPU)

## 1. 模块概述

向量浮点单元 (Vector Floating Point Unit, VFPU) 是 OpenC910 处理器的浮点和向量运算核心，支持 IEEE 754 浮点标准和 RISC-V 向量扩展。VFPU 包含浮点 ALU、浮点乘加单元和向量运算单元，可执行标量浮点运算和向量 SIMD 运算。

### 1.1 主要功能

- 浮点加减运算
- 浮点乘法
- 浮点乘加 (FMA)
- 浮点除法/开方
- 浮点比较
- 浮点格式转换
- 向量运算

## 5. 浮点格式支持

### 5.1 支持的格式

| 格式 | 位宽 | 描述 |
|-----|------|------|
| FP16 | 16 | 半精度浮点 |
| FP32 | 32 | 单精度浮点 |
| FP64 | 64 | 双精度浮点 |

### 5.2 IEEE 754 兼容性

- 支持所有 IEEE 754 舍入模式
- 支持非规格化数处理
- 支持无穷大和 NaN 处理

## 6. 舍入模式

| 模式 | 编码 | 描述 |
|-----|------|------|
| RNE | 000 | 向偶数舍入 |
| RTZ | 001 | 向零舍入 |
| RDN | 010 | 向下舍入 |
| RUP | 011 | 向上舍入 |
| RMM | 100 | 向最近舍入 |
| DYN | 111 | 动态舍入 |

## 7. 浮点异常

| 异常 | 标志 | 描述 |
|-----|------|------|
| NV | 0 | 无效操作 |
| DZ | 1 | 除零 |
| OF | 2 | 上溢 |
| UF | 3 | 下溢 |
| NX | 4 | 不精确 |

## 8. 向量扩展支持

### 8.1 向量寄存器

- 32 个向量寄存器 (v0-v31)
- 可配置向量长度 (VLEN)
- 支持向量分组

### 8.2 向量指令类型

| 类型 | 描述 |
|-----|------|
| 向量-向量 | 两个向量操作数 |
| 向量-标量 | 向量和标量操作数 |
| 向量-立即数 | 向量和立即数操作数 |

## 9. 设计要点

1. **IEEE 754 兼容**: 完全符合 IEEE 754 标准
2. **流水线设计**: 多级流水线提高吞吐量
3. **向量支持**: 完整支持 RISC-V V 扩展
4. **低延迟**: 优化的浮点运算延迟
