# ct_lsu_top 模块设计文档

## 1. 模块概述

### 1.1 基本信息

| 属性 | 值 |
|------|-----|
| 模块名称 | ct_lsu_top |
| 文件路径 | lsu/rtl/ct_lsu_top.v |
| 功能描述 | 访存单元顶层模块 |
| 设计特点 | 支持DCache、写缓冲、内存一致性 |

### 1.2 功能描述

ct_lsu_top 是 OpenC910 处理器的访存单元，负责处理所有内存访问操作。主要功能包括：

- **加载存储**：处理Load/Store指令
- **DCache管理**：数据缓存管理
- **地址翻译**：虚拟地址到物理地址转换
- **内存一致性**：维护缓存一致性

### 1.3 设计特点

- 支持乱序访存
- 支持写缓冲合并
- 支持非对齐访问
- 支持原子操作
