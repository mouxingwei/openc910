# OpenC910 一级模块文档：整数执行单元 (IU)

## 1. 模块概述

整数执行单元 (Integer Unit, IU) 是 OpenC910 处理器的整数运算核心，负责所有整数运算、分支执行和乘除法操作。IU 包含多个 ALU、乘法器、除法器和分支单元，支持超标量执行。

### 1.1 主要功能

- 整数加减运算
- 逻辑运算
- 移位操作
- 乘法运算
- 除法运算
- 分支条件计算
- 跳转地址计算

## 5. 执行流水线

### 5.1 Pipe0 (ALU 短操作)

```
EX1         EX2
┌─────────┬─────────┐
│ ALU运算 │ 结果写回│
│ 1周期   │ 1周期   │
└─────────┴─────────┘
```

支持的指令:
- ADD, ADDI, ADDIW
- AND, ANDI
- OR, ORI
- XOR, XORI
- SLL, SLLI
- SRL, SRLI
- SRA, SRAI
- SLT, SLTI
- SLTU, SLTIU

### 5.2 Pipe1 (ALU + 乘法)

```
EX1         EX2         EX3
┌─────────┬─────────┬─────────┐
│ ALU/MULT│ 乘法继续│ 结果写回│
│ 1周期   │ 1周期   │ 1周期   │
└─────────┴─────────┴─────────┘
```

支持的指令:
- 所有 Pipe0 支持的指令
- MUL, MULH, MULHSU, MULHU
- MULW

### 5.3 Pipe2 (分支/跳转)

```
EX1         EX2
┌─────────┬─────────┐
│ 条件判断│ 目标计算│
│ PC计算  │ 结果处理│
└─────────┴─────────┘
```

支持的指令:
- BEQ, BNE, BLT, BGE, BLTU, BGEU
- JAL, JALR
- AUIPC

### 5.4 除法器 (非流水)

```
迭代周期: 8-16 周期 (取决于操作数)
```

支持的指令:
- DIV, DIVU
- DIVW, DIVUW
- REM, REMU
- REMW, REMUW

## 6. 功能单元详解

### 6.1 ALU 功能

| 功能码 | 操作 | 描述 |
|-------|------|------|
| ADD | a + b | 加法 |
| SUB | a - b | 减法 |
| AND | a & b | 按位与 |
| OR | a \| b | 按位或 |
| XOR | a ^ b | 按位异或 |
| SLL | a << b | 逻辑左移 |
| SRL | a >> b | 逻辑右移 |
| SRA | a >>> b | 算术右移 |
| SLT | a < b (有符号) | 小于比较 |
| SLTU | a < b (无符号) | 无符号小于比较 |

### 6.2 乘法器特性

- 65x65 位乘法
- 3 级流水线
- 支持有符号/无符号
- 支持高/低位结果

### 6.3 除法器特性

- SRT-Radix16 算法
- 非流水执行
- 支持有符号/无符号
- 支持商/余数结果

## 7. 分支预测验证

### 7.1 PC FIFO

PC FIFO 存储分支预测信息:
- 当前 PC
- 预测目标 PC
- 预测方向
- 检查索引

### 7.2 预测验证流程

```
1. BJU 执行分支指令
2. 计算实际目标地址
3. 与预测目标比较
4. 如不匹配，触发预测错误
5. 通知 IFU 重定向
```

## 8. 设计要点

1. **超标量执行**: 多条指令并行执行
2. **前递支持**: 支持结果前递到 IDU
3. **分支验证**: 实时验证分支预测
4. **除法优化**: SRT-16 算法加速除法
