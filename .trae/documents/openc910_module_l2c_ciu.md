# OpenC910 一级模块文档：二级缓存与互连 (L2C & CIU)

## 1. 模块概述

二级缓存与互连单元 (L2 Cache & Cache Interconnect Unit) 是 OpenC910 处理器的共享缓存和核间互连系统。L2C 提供大容量共享缓存，CIU 负责多核之间的缓存一致性维护和数据互连。

### 1.1 主要功能

- L2 缓存管理
- 多核缓存一致性
- 核间数据互连
- 缓存行填充与回写
- Snoop 协议处理

## 6. 缓存一致性协议

### 6.1 MESI 状态

| 状态 | 描述 |
|-----|------|
| Modified | 已修改，独占拥有最新数据 |
| Exclusive | 独占，干净数据 |
| Shared | 共享，干净数据 |
| Invalid | 无效 |

### 6.2 状态转换

```
                读请求
Invalid ──────────────► Shared
    │                      │
    │ 写请求               │ 写请求
    ▼                      ▼
Modified ◄──────────── Exclusive
    │
    │ Snoop请求
    ▼
Shared (写回数据)
```

## 7. 目录协议

### 7.1 目录结构

| 字段 | 描述 |
|-----|------|
| Tag | 地址标签 |
| State | MESI状态 |
| Sharers | 共享者位图 |
| Owner | 拥有者ID |

### 7.2 目录操作

- **读请求**: 检查目录，返回数据或转发
- **写请求**: 使其他副本无效
- **Snoop**: 更新目录状态

## 8. 性能优化

### 8.1 预取

- 顺序预取
- 跳跃预取

### 8.2 写合并

- 合并相同缓存行的写操作
- 减少总线事务

## 9. 设计要点

1. **大容量缓存**: 支持大容量 L2 缓存
2. **多核一致性**: 完整的 MESI 协议支持
3. **目录协议**: 高效的多核一致性维护
4. **低延迟**: 优化的缓存访问路径
