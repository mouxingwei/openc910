# OpenC910 二级模块文档：分支历史表 (BHT)

## 1. 模块概述

分支历史表 (Branch History Table, BHT) 是 IFU 中分支预测器的重要组成部分，用于预测条件分支的跳转方向。BHT 采用 2-bit 饱和计数器来记录分支的历史行为。

### 1.1 所属关系

```
IFU (一级模块)
├── 分支预测子系统
│   ├── L0-BTB (二级模块)
│   ├── BTB (二级模块)
│   ├── BHT (二级模块) ← 本文档
│   └── RAS (二级模块)
```

## 2. 模块列表

| 模块名称 | 文件 | 功能描述 |
|---------|------|---------|
| ct_ifu_bht | ct_ifu_bht.v | BHT 顶层 |
| ct_ifu_bht_pre_array | ct_ifu_bht_pre_array.v | BHT 预测阵列 |
| ct_ifu_bht_sel_array | ct_ifu_bht_sel_array.v | BHT 选择阵列 |

## 3. 配置参数

| 参数 | 值 |
|-----|-----|
| 容量 | 4096-8192 项 |
| 预测器类型 | 2-bit 饱和计数器 |
| 索引方式 | PC 低位索引 |

## 4. 2-bit 饱和计数器

### 4.1 状态定义

| 状态 | 编码 | 预测方向 | 描述 |
|-----|------|---------|------|
| Strongly Not Taken | 00 | 不跳转 | 强不跳转 |
| Weakly Not Taken | 01 | 不跳转 | 弱不跳转 |
| Weakly Taken | 10 | 跳转 | 弱跳转 |
| Strongly Taken | 11 | 跳转 | 强跳转 |

### 4.2 状态转换

```
              跳转
    ┌─────────────────────┐
    ▼                     │
┌───────┐ 跳转   ┌───────┐
│  00   │───────▶│  01   │
│ SN    │◀───────│ WN    │
└───────┘ 不跳转 └───────┘
    ▲                     │
    │      跳转           │
    │    ┌───────┐        │
    │    │  10   │◀───────┘
    │    │ WT    │
    │    └───────┘
    │         │ 不跳转
    │         ▼
    │    ┌───────┐
    └────│  11   │
         │ ST    │
         └───────┘
              │ 不跳转
              ▼
         ┌───────┐
         │  10   │
         └───────┘
```

## 5. 阵列结构

### 5.1 预测阵列

```
┌─────────────────────────────────────────┐
│ Entry[0] │ Entry[1] │ ... │ Entry[N-1] │
│  2-bit   │  2-bit   │     │   2-bit    │
└─────────────────────────────────────────┘
     ▲
     │
   PC[log2(N):1] 索引
```

### 5.2 选择阵列

用于存储额外的选择信息，支持更复杂的预测算法。

## 6. 操作流程

### 6.1 预测流程

```
1. 取指 PC 输入 BHT
2. 使用 PC 低位索引预测阵列
3. 读取 2-bit 计数器值
4. 根据最高位确定预测方向:
   - MSB = 1: 预测跳转
   - MSB = 0: 预测不跳转
```

### 6.2 更新流程

```
1. 分支指令执行完成
2. 获取实际跳转方向
3. 读取当前计数器值
4. 根据实际方向更新:
   - 跳转: 计数器 +1 (饱和在 11)
   - 不跳转: 计数器 -1 (饱和在 00)
```

## 7. 接口信号

### 7.1 预测接口

| 信号名称 | 方向 | 位宽 | 功能描述 |
|---------|------|------|---------|
| bht_pc | I | 40 | 查询 PC |
| bht_en | I | 1 | 查询使能 |
| bht_pred | O | 1 | 预测方向 (1=跳转) |
| bht_conf | O | 1 | 预测置信度 |

### 7.2 更新接口

| 信号名称 | 方向 | 位宽 | 功能描述 |
|---------|------|------|---------|
| bht_update_en | I | 1 | 更新使能 |
| bht_update_pc | I | 40 | 更新 PC |
| bht_update_taken | I | 1 | 实际跳转方向 |

## 8. 与 BTB 的协作

```
分支预测流程:
1. BTB 提供分支目标地址
2. BHT 提供跳转方向预测
3. 组合结果:
   - BTB 命中 + BHT 预测跳转 = 使用 BTB 目标
   - BTB 命中 + BHT 预测不跳转 = 顺序取指
   - BTB 未命中 = 顺序取指
```

## 9. 设计要点

1. **简单高效**: 2-bit 计数器实现简单
2. **自适应**: 根据历史行为自适应调整
3. **低延迟**: 单周期预测
4. **高准确率**: 对大多数分支模式有良好预测准确率
