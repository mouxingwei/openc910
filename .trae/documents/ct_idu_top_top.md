# ct_idu_top 模块设计文档

## 1. 模块概述

### 1.1 基本信息

| 属性 | 值 |
|------|-----|
| 模块名称 | ct_idu_top |
| 文件路径 | idu/rtl/ct_idu_top.v |
| 功能描述 | 译码单元顶层模块 |
| 设计特点 | 支持多发射译码、指令分发、依赖检查 |

### 1.2 功能描述

ct_idu_top 是 OpenC910 处理器的译码单元，负责将取指单元送来的指令进行译码并分发给各执行单元。主要功能包括：

- **指令译码**：解析指令的操作码和操作数
- **依赖检查**：检测指令间的数据依赖
- **指令分发**：将指令分发到合适的执行单元
- **寄存器重命名**：支持乱序执行的重命名机制

### 1.3 设计特点

- 支持双发射译码
- 支持动态调度
- 支持Scoreboard机制
- 支持指令融合优化
