# ct_vfpu_top 模块设计文档

## 1. 模块概述

### 1.1 基本信息

| 属性 | 值 |
|------|-----|
| 模块名称 | ct_vfpu_top |
| 文件路径 | vfpu/rtl/ct_vfpu_top.v |
| 功能描述 | 向量浮点单元顶层模块 |
| 设计特点 | 支持向量运算、浮点运算、SIMD |

### 1.2 功能描述

ct_vfpu_top 是 OpenC910 处理器的向量浮点单元，负责执行浮点运算和向量运算。主要功能包括：

- **浮点运算**：浮点加减乘除
- **向量运算**：SIMD向量操作
- **类型转换**：整数与浮点转换
- **特殊处理**：NaN、Inf等特殊值处理

### 1.3 设计特点

- 支持IEEE 754标准
- 支持RISC-V向量扩展
- 多级流水线设计
- 支持Fused Multiply-Add
