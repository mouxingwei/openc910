# OpenC910 总体架构方案

## 1. 概述

OpenC910 是平头哥半导体(T-Head Semiconductor)开源的高性能 RISC-V 64位处理器核心。该处理器采用乱序执行、超标量架构设计，支持双核 SMP 配置，具有完整的内存管理、多级缓存系统和丰富的调试接口。

### 1.1 核心特性

| 特性 | 描述 |
|-----|------|
| 架构 | RISC-V 64位 ISA |
| 架构特性 | 乱序执行、超标量 |
| 流水线深度 | 10-12级流水线 |
| 发射宽度 | 3发射（每周期最多发射3条指令） |
| 提交宽度 | 3提交（每周期最多提交3条指令） |
| 核心配置 | 支持双核 SMP |
| 缓存配置 | I-Cache + D-Cache + L2 Cache |
| 内存管理 | SV39/SV48 虚拟内存支持 |
| 扩展支持 | 向量扩展(V)、浮点扩展(F/D) |

### 1.2 支持的指令集扩展

- **RV64GC**: 通用计算扩展 (IMAFD)
- **RV64V**: 向量扩展
- **自定义扩展**: T-Head 扩展指令集

## 3. 一级模块划分

### 3.1 核心处理模块

| 模块名称 | 缩写 | 功能描述 |
|---------|------|---------|
| 指令取指单元 | IFU | 取指、分支预测、指令缓存 |
| 指令分发单元 | IDU | 指令解码、寄存器重命名、发射队列 |
| 整数执行单元 | IU | ALU、乘法器、除法器、分支单元 |
| 浮点/向量单元 | VFPU | 浮点运算、向量运算 |
| 访存单元 | LSU | 数据缓存、访存流水线 |
| 提交单元 | RTU | 重排序缓冲区、指令提交 |

### 3.2 系统控制模块

| 模块名称 | 缩写 | 功能描述 |
|---------|------|---------|
| 内存管理单元 | MMU | TLB、地址翻译、页表遍历 |
| 协处理器0 | CP0 | CSR寄存器、异常处理 |
| 物理内存保护 | PMP | 内存区域权限控制 |
| 总线接口单元 | BIU | AXI总线接口、协议转换 |

### 3.3 调试与监控模块

| 模块名称 | 缩写 | 功能描述 |
|---------|------|---------|
| 硬件辅助调试 | HAD | JTAG调试、断点、跟踪 |
| 硬件性能计数器 | HPCP | 性能事件统计 |

### 3.4 系统互连模块

| 模块名称 | 缩写 | 功能描述 |
|---------|------|---------|
| 缓存互连单元 | CIU | 多核缓存一致性、总线互连 |
| 二级缓存 | L2C | 共享二级缓存 |
| 核心本地中断器 | CLINT | 软件中断、定时器中断 |
| 平台级中断控制器 | PLIC | 外部中断管理 |

## 4. 流水线架构

### 4.1 流水线阶段

```
┌──────────────────────────────────────────────────────────────────────────┐
│                           OpenC910 流水线                                 │
├──────────┬──────────┬──────────┬──────────┬──────────┬──────────┬───────┤
│   IF1    │   IF2    │   ID     │   IS     │   EX     │   WB     │  RET  │
│  取指1   │  取指2   │  译码    │  发射    │  执行    │  写回    │  提交 │
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────┼───────┤
│ PC生成   │ I-Cache  │ 指令解码 │ 发射队列 │ ALU/LSU  │ 结果写回  │ ROB   │
│ 分支预测 │ 指令缓冲 │ 寄存器   │ 操作数   │ 乘除法   │ 寄存器    │ 提交  │
│ BTB查找  │ 预解码   │ 重命名   │ 就绪检查 │ 浮点运算 │ 更新      │ 异常  │
└──────────┴──────────┴──────────┴──────────┴──────────┴──────────┴───────┘
```

### 4.2 执行流水线

| 流水线 | 功能 | 执行单元 |
|-------|------|---------|
| Pipe0 | ALU短操作 | ALU、逻辑运算 |
| Pipe1 | ALU/乘法 | ALU、乘法器 |
| Pipe2 | 分支/跳转 | BJU、PC计算 |
| Pipe3 | Load操作 | LSU Load通路 |
| Pipe4 | Store操作 | LSU Store通路 |
| Pipe5 | 向量访存 | 向量LSU |
| Pipe6 | 浮点ALU | VFALU |
| Pipe7 | 浮点乘加 | VFMAU |

## 7. 中断与异常处理

### 7.1 中断源

| 中断类型 | 来源 | 处理模块 |
|---------|------|---------|
| 软件中断 | CLINT | CLINT → CP0 |
| 定时器中断 | CLINT | CLINT → CP0 |
| 外部中断 | PLIC | PLIC → CP0 |
| 调试中断 | HAD | HAD → RTU |

### 7.2 异常处理流程

```
异常发生 → RTU捕获 → CP0记录 → IFU重定向 → 异常处理程序
```

## 8. 调试支持

### 8.1 调试接口

- **JTAG**: 标准JTAG调试接口

### 8.2 调试功能

### 9.2 电源状态

| 状态 | 描述 |
