# 寄存器手册生成

## 功能概述

本SKILL根据模块详细方案文档生成寄存器手册。

## 使用场景

- 模块设计完成，需要生成寄存器手册
- 需要定义寄存器的地址映射
- 需要详细描述寄存器字段

## 工作流程

### Step 1: 收集输入

1. 读取模块详细方案文档中的寄存器相关章节
2. 确认需要生成的寄存器列表
3. 了解寄存器的访问属性

### Step 2: 分析寄存器信息

从方案中提取以下信息：

**寄存器信息**：
- 寄存器名称
- 寄存器地址
- 访问属性（RO/RW/WO/W1C等）
- 复位值
- 描述

**字段信息**：
- 字段名称
- 字段位置
- 字段属性
- 字段描述
- 字段值含义

### Step 3: 生成寄存器手册

按以下结构生成文档：

## 3. 寄存器详细定义

### 3.1 CTRL - 控制寄存器 (0x00)

**描述**：模块功能控制寄存器

**访问类型**：RW

**复位值**：0x00000000

**位定义**：

| 位 | 名称 | 访问 | 复位 | 描述 |
|----|------|------|------|------|
| [31:16] | RESERVED | RO | 0x0 | 保留 |
| [15:8] | MODE | RW | 0x0 | 工作模式 |
| [7:4] | RESERVED | RO | 0x0 | 保留 |
| [3] | ENABLE | RW | 0x0 | 模块使能：1=使能，0=关闭 |
| [2] | INT_CLR | W1C | 0x0 | 中断清除：写1清除 |
| [1] | INT_SET | WO | - | 中断设置：写1设置 |
| [0] | START | RW | 0x0 | 启动：1=启动工作 |

**字段详细描述**：

- **MODE [15:8]**：工作模式选择
  - 0x0：正常模式
  - 0x1：低功耗模式
  - 0x2：测试模式

- **ENABLE [3]**：模块使能控制
  - 0：模块关闭
  - 1：模块使能

### 3.2 STATUS - 状态寄存器 (0x04)

**描述**：模块状态寄存器

**访问类型**：RO

**复位值**：0x00000000

**位定义**：

| 位 | 名称 | 访问 | 复位 | 描述 |
|----|------|------|------|------|
| [31:8] | RESERVED | RO | 0x0 | 保留 |
| [7:4] | RESERVED | RO | 0x0 | 保留 |
| [3] | BUSY | RO | 0x0 | 忙标志：1=工作中 |
| [2] | DONE | RO | 0x0 | 完成标志：1=完成 |
| [1] | ERROR | RO | 0x0 | 错误标志：1=有错误 |
| [0] | INT_PEND | RO | 0x0 | 中断挂起：1=有中断 |

### 3.3 INT_EN - 中断使能寄存器 (0x08)

**描述**：中断使能寄存器

**访问类型**：RW

**复位值**：0x00000000

**位定义**：

| 位 | 名称 | 访问 | 复位 | 描述 |
|----|------|------|------|------|
| [31:4] | RESERVED | RO | 0x0 | 保留 |
| [3] | EN_DONE | RW | 0x0 | 完成中断使能 |
| [2] | EN_ERROR | RW | 0x0 | 错误中断使能 |
| [1] | RESERVED | RO | 0x0 | 保留 |
| [0] | EN_TIMEOUT | RW | 0x0 | 超时中断使能 |

## 4. 寄存器访问示例

### 4.1 初始化序列

```c
// 初始化模块
void module_init(void *base) {
    // 写控制寄存器，使能模块
    writel(base + CTRL_OFFSET, 0x1);
    
    // 使能中断
    writel(base + INT_EN_OFFSET, 0xF);
}
```

### 4.2 状态查询

```c
// 查询模块状态
uint32_t get_status(void *base) {
    return readl(base + STATUS_OFFSET);
}
```

### Step 4: 质量检查

**完整性检查**：
- [ ] 所有寄存器都有定义

**一致性检查**：
- [ ] 地址不重叠
- [ ] 字段不冲突
- [ ] 复位值正确

## 注意事项

1. **定义完整**：每个寄存器都要有完整的定义
2. **描述清晰**：字段描述要清晰明确
3. **访问正确**：访问属性要准确
4. **示例充分**：提供足够的访问示例
