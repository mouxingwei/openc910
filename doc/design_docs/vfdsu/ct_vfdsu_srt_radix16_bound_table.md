# ct_vfdsu_srt_radix16_bound_table 模块详细设计方案

## 1. 模块概述

### 1.1 模块名称
`ct_vfdsu_srt_radix16_bound_table` - SRT radix-16边界值查找表模块

### 1.2 功能描述
`ct_vfdsu_srt_radix16_bound_table` 实现SRT radix-16算法中的商位选择边界值查找表。该模块根据除数（除法）或部分根（平方根）的高位值，输出用于商位选择的边界值。

### 1.3 主要特性
- 64组边界值查找表
- 支持除法和平方根运算
- 支持平方根首轮和次轮特殊处理
- 组合逻辑实现，单周期输出

## 2. SRT商位选择原理

### 2.1 边界值作用
在SRT算法中，商位的选择基于余数与边界值的比较。边界值由除数（除法）或部分根（平方根）决定。

### 2.2 商位选择规则

```
对于radix-16 SRT，商位q的选择规则:

如果 remainder >= bound[+k]，则 q = +k
如果 remainder <  bound[-k]，则 q = -k
否则根据具体边界值确定

其中 k ∈ {1, 2, ..., 9}
```

### 2.3 边界值计算
边界值是归一化除数的函数，预先计算并存储在查找表中。

## 5. 查找表结构

### 5.1 边界选择值范围
`bound_sel` 的取值范围为 7'h40 到 7'h00，共64个可能值。

### 5.2 边界值格式
每个边界值为7位，表示归一化的比较阈值。

### 5.3 查找表实现

```verilog
always @(*) begin
    case (bound_sel)
        7'h40: begin
            bound_1 = 7'hxx;
            bound_2 = 7'hxx;
            // ...
            bound_9 = 7'hxx;
        end
        7'h3F: begin
            bound_1 = 7'hxx;
            bound_2 = 7'hxx;
            // ...
            bound_9 = 7'hxx;
        end
        // ... 其他64个case
        default: begin
            bound_1 = 7'h00;
            bound_2 = 7'h00;
            // ...
            bound_9 = 7'h00;
        end
    endcase
end
```

## 6. 平方根特殊处理

### 6.1 首轮特殊边界
平方根运算的第一轮迭代使用特殊的边界值，因为此时部分根为零。

```verilog
// 平方根首轮边界值
assign sqrt_first_bound_1 = 7'hxx;
assign sqrt_first_bound_2 = 7'hxx;
// ...
```

### 6.2 次轮特殊边界
平方根运算的第二轮迭代也使用特殊边界值。

```verilog
// 平方根次轮边界值
assign sqrt_second_bound_1 = 7'hxx;
assign sqrt_second_bound_2 = 7'hxx;
// ...
```

### 6.3 边界值选择逻辑

```verilog
always @(*) begin
    if (sqrt_first_round) begin
        // 使用首轮特殊边界值
        bound_1 = sqrt_first_bound_1;
        bound_2 = sqrt_first_bound_2;
        // ...
    end else if (sqrt_second_round) begin
        // 使用次轮特殊边界值
        bound_1 = sqrt_second_bound_1;
        bound_2 = sqrt_second_bound_2;
        // ...
    end else begin
        // 使用正常查找表
        // ...
    end
end
```

## 7. 商位选择示例

### 7.1 除法商位选择

```
假设:
- 余数高位: rem[60:54] = 7'h20
- 除数高位: bound_sel = 7'h30
- 查表得: bound_1=0x10, bound_2=0x20, bound_3=0x30

比较:
- rem >= bound_1 (0x20 >= 0x10) ✓
- rem >= bound_2 (0x20 >= 0x20) ✓
- rem >= bound_3 (0x20 >= 0x30) ✗

结果: q = +2
```

### 7.2 平方根商位选择

```
平方根的边界值基于部分根Q的高位:
- bound_sel = Q[6:0]
- 其余逻辑与除法类似
```

## 8. 边界值计算公式

### 8.1 除法边界值
```
bound_k = floor(16 * k / d)

其中:
- k: 商位值 (1~9)
- d: 归一化除数
```

### 8.2 平方根边界值
```
bound_k = floor(k * (2Q + k*16^(-i)))

其中:
- k: 根位值 (1~9)
- Q: 当前部分根
- i: 迭代次数
```

## 9. 设计要点

### 9.1 查找表优化
- 使用case语句实现，综合后为组合逻辑
- 可考虑使用ROM实现以减少面积

### 9.2 时序考虑
- 查找表为纯组合逻辑
- 输出直接用于下一级比较器

### 9.3 精度保证
- 边界值精度影响商位选择准确性
- 需要保证边界值计算的舍入方向正确
