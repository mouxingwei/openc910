# ct_vfdsu_round 模块详细设计方案

## 1. 模块概述

### 1.1 模块名称
`ct_vfdsu_round` - 浮点舍入处理模块

### 1.2 功能描述
`ct_vfdsu_round` 是VFDSU流水线EX3阶段的模块，负责对SRT迭代产生的商/根进行IEEE 754标准的舍入处理。该模块支持五种舍入模式，处理舍入进位、非规格化数舍入以及溢出/下溢情况。

### 1.3 主要特性
- IEEE 754标准舍入
- 支持五种舍入模式（RNE、RTZ、RDN、RUP、RMM）
- Guard/Sticky位处理
- 非规格化数舍入
- 舍入进位传播
- 溢出/下溢标志更新

## 2. IEEE 754舍入模式

### 2.1 舍入模式定义

| 编码 | 模式 | 名称 | 描述 |
|------|------|------|------|
| 000 | RNE | Round to Nearest Even | 向最近偶数舍入 |
| 001 | RTZ | Round Toward Zero | 向零舍入 |
| 010 | RDN | Round Down | 向负无穷舍入 |
| 011 | RUP | Round Up | 向正无穷舍入 |
| 100 | RMM | Round to Max Magnitude | 向最大幅值舍入 |

### 2.2 舍入位定义

```
对于双精度:
| 有效位 [51:0] | Guard | Round | Sticky |
|     52位       |  1位   |  1位  |   1位  |

- Guard位 (G): 被舍弃的最高位
- Round位 (R): Guard位的下一位
- Sticky位 (S): R位以下所有位的或值
```

### 2.3 舍入决策

```
RNE: 如果G=0，截断；如果G=1且(R=1或S=1)，进位；如果G=1且R=S=0，向偶数舍入
RTZ: 总是截断（向零方向）
RDN: 负数时截断，正数时若有舍入位则减1
RUP: 正数时截断，负数时若有舍入位则减1
RMM: 如果G=1，进位；否则截断
```

## 5. 关键逻辑设计

### 5.1 舍入位提取

```verilog
// 双精度舍入位提取
// qt格式: [55:52]扩展位, [51:0]有效位, [52]G, [53]R, [54]S

assign qt_gr = ex2_qt[52];           // Guard位
assign qt_r = ex2_qt[53];            // Round位
assign qt_s = ex2_qt[54] | ex2_qt[55] | (!ex2_rem_zero) | ex2_rem_sign;  // Sticky位

// 有效位
assign qt_frac[51:0] = ex2_qt[51:0];
assign qt_lsb = ex2_qt[0];           // 最低位（用于向偶舍入）
```

### 5.2 舍入决策

#### 5.2.1 RNE (Round to Nearest Even)
```verilog
// 向最近偶数舍入
// 如果G=1且(R=1或S=1)，进位
// 如果G=1且R=S=0，向偶数舍入（LSB=1时进位）
assign frac_rne_add_1 = qt_gr && (qt_r || qt_s || qt_lsb);
```

#### 5.2.2 RTZ (Round Toward Zero)
```verilog
// 向零舍入，总是截断
assign frac_rtz_add_1 = 1'b0;
```

#### 5.2.3 RDN (Round Down)
```verilog
// 向负无穷舍入
// 负数时，如果有舍入位则减1（相当于加-1）
assign frac_rdn_add_1 = sign && (qt_gr || qt_r || qt_s);
```

#### 5.2.4 RUP (Round Up)
```verilog
// 向正无穷舍入
// 正数时，如果有舍入位则进位
assign frac_rup_add_1 = !sign && (qt_gr || qt_r || qt_s);
```

#### 5.2.5 RMM (Round to Max Magnitude)
```verilog
// 向最大幅值舍入
// 如果G=1，进位
assign frac_rmm_add_1 = qt_gr;
```

### 5.3 舍入模式选择

```verilog
always @(*) begin
    case (rm)
        3'b000: round_add_1 = frac_rne_add_1;  // RNE
        3'b001: round_add_1 = frac_rtz_add_1;  // RTZ
        3'b010: round_add_1 = frac_rdn_add_1;  // RDN
        3'b011: round_add_1 = frac_rup_add_1;  // RUP
        3'b100: round_add_1 = frac_rmm_add_1;  // RMM
        default: round_add_1 = 1'b0;
    endcase
end
```

### 5.4 舍入执行

```verilog
// 舍入加法
assign {qt_carry, qt_frac_rnd} = {1'b0, qt_frac} + round_add_1;

// 指数调整
assign expt_rnd = ex2_expt + qt_carry;

// 舍入后尾数
assign ex3_qt_rnd = {qt_carry, qt_frac_rnd};
```

### 5.5 非规格化数舍入

```verilog
// 非规格化数检测
assign is_denorm = (expt_rnd == 0);

// 非规格化舍入处理
// 如果结果是非规格化数，需要右移尾数
always @(*) begin
    if (is_denorm && ex2_underflow) begin
        // 非规格化数舍入
        denorm_shift = 1 - expt_rnd;
        qt_frac_denorm = qt_frac >> denorm_shift;
        // 重新计算舍入位
        // ...
    end else begin
        qt_frac_denorm = qt_frac_rnd;
    end
end
```

### 5.6 溢出/下溢更新

```verilog
// 舍入后重新检测溢出
assign ex3_overflow = ex2_overflow || 
                      (expt_rnd > MAX_EXPT);

// 舍入后重新检测下溢
assign ex3_underflow = ex2_underflow || 
                       (expt_rnd < MIN_EXPT);
```

## 6. 舍入示例

### 6.1 RNE舍入示例

```
输入: qt = 1.0101100...0 | 1 | 0 | 0
      (有效位以0结尾，G=1，R=S=0)

分析: G=1，R=S=0，LSB=0
      向偶数舍入，LSB=0，不进位

结果: qt_rnd = 1.0101100...0
```

```
输入: qt = 1.0101100...1 | 1 | 0 | 0
      (有效位以1结尾，G=1，R=S=0)

分析: G=1，R=S=0，LSB=1
      向偶数舍入，LSB=1，进位

结果: qt_rnd = 1.0101101...0
```

### 6.2 RTZ舍入示例

```
输入: qt = 1.0101100...0 | 1 | 1 | 1

分析: RTZ总是截断

结果: qt_rnd = 1.0101100...0
```

### 6.3 RDN舍入示例

```
输入: qt = 1.0101100...0 | 1 | 0 | 0, sign = 1 (负数)

分析: 负数，G=1，需要减1（向下舍入）

结果: qt_rnd = 1.0101011...1 (减1后)
```

### 9.2 非规格化处理
- 非规格化数需要特殊舍入
- 可能需要多次右移

- 舍入可能导致溢出
- 需要重新检测
