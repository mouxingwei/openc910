# cpu_cfig.h 配置文档

## 1. 模块概述

### 1.1 基本信息
| 项目 | 内容 |
|------|------|
| 文件名称 | cpu_cfig.h |
| 文件路径 | C910_RTL_FACTORY/gen_rtl/cpu/rtl/cpu_cfig.h |
| 文件类型 | 配置头文件 |
| 作者 | T-Head Semiconductor Co., Ltd. |
| 许可证 | Apache License 2.0 |

### 1.2 功能描述
cpu_cfig.h 是 OpenC910 处理器的配置头文件，定义了处理器的各种配置宏，包括版本信息、缓存大小、TLB 条目数、中断配置等关键参数。

### 1.3 设计特点
- 集中配置管理
- 编译时可配置
- 支持多种配置选项

## 2. 配置宏定义

### 2.1 版本信息

| 宏名称 | 值 | 描述 |
|--------|-----|------|
| REVISION | 1.0 | 版本号 |
| PRODUCT_ID | 0xC910 | 产品 ID |

### 2.2 缓存配置

| 宏名称 | 值 | 描述 |
|--------|-----|------|
| ICACHE_64K | 1 | I-Cache 大小为 64KB |
| DCACHE_64K | 1 | D-Cache 大小为 64KB |
| L2_CACHE_1M | 1 | L2 Cache 大小为 1MB |

### 2.3 TLB 配置

| 宏名称 | 值 | 描述 |
|--------|-----|------|
| JTLB_ENTRY_1024 | 1 | JTLB 条目数为 1024 |

### 2.4 中断配置

| 宏名称 | 值 | 描述 |
|--------|-----|------|
| PLIC_INTNUM | 144 | PLIC 中断数量 |

### 2.5 多处理器配置

| 宏名称 | 值 | 描述 |
|--------|-----|------|
| MP_EN | 1 | 多处理器使能 |

## 3. 配置说明

### 3.1 缓存大小配置
处理器支持以下缓存配置：
- I-Cache: 64KB
- D-Cache: 64KB
- L2 Cache: 1MB (共享)

### 3.2 TLB 配置
- JTLB (联合 TLB): 1024 条目
- 支持多级页表遍历

### 3.3 中断配置
- PLIC 支持 144 个外部中断源
- 支持中断优先级
- 支持中断路由

### 3.4 多处理器配置
- 支持双核 SMP 配置
- 共享 L2 缓存
- 缓存一致性协议

## 4. 使用说明

### 4.1 配置修改
修改配置宏可以改变处理器的配置：
1. 打开 cpu_cfig.h 文件
2. 修改相应的宏定义
3. 重新编译 RTL 代码

### 4.2 配置依赖
某些配置之间存在依赖关系：
- L2 缓存配置依赖多处理器配置
- TLB 条目数影响地址空间覆盖

## 5. 设计注意事项

### 5.1 配置验证
修改配置后需要验证：
- 时序收敛
- 资源利用率
- 功能正确性

### 5.2 配置限制
某些配置有最大/最小限制：
- 缓存大小必须是 2 的幂次
- TLB 条目数必须是 2 的幂次
