### 1.2 功能描述

ct_idu_is_lsiq是IDU（指令分发单元）发射阶段的加载存储发射队列，负责：

1. **访存指令缓存**：缓存等待发射的加载和存储指令
2. **发射仲裁**：仲裁就绪访存指令的发射顺序
3. **地址相关性检查**：检查访存地址的相关性
4. **数据前递**：支持数据前递机制
5. **队列管理**：管理队列的创建、发射和清空

## 4. 模块实现方案

### 4.1 队列结构

LSIQ采用表项结构，每个表项存储一条访存指令的信息：
- 访存类型（加载/存储）
- 访存地址
- 数据大小
- 源操作数信息

### 4.2 关键逻辑

- 创建逻辑：接收访存指令，分配表项
- 发射仲裁：选择最老的就绪访存指令
- 地址相关性检查：检查访存地址冲突
- 旁路逻辑：支持访存数据旁路
