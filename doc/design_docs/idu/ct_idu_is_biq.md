### 1.2 功能描述

ct_idu_is_biq是IDU（指令分发单元）发射阶段的分支发射队列，负责：

1. **分支指令缓存**：缓存等待发射的分支指令
2. **发射仲裁**：仲裁就绪分支指令的发射顺序
3. **相关性检查**：检查分支指令源操作数的相关性
4. **数据前递**：支持数据前递机制
5. **队列管理**：管理队列的创建、发射和清空

### 1.3 设计特点

- 专门处理分支指令
- 支持条件分支和无条件分支
- 高效的发射仲裁机制
- 支持数据旁路和前递
- 低功耗设计（门控时钟）

## 4. 模块实现方案

### 4.1 队列结构

BIQ采用表项结构，每个表项存储一条分支指令的信息：
- 分支类型（条件/无条件）
- 分支目标地址
- 分支条件
- 源操作数信息

### 4.2 关键逻辑

- 创建逻辑：接收分支指令，分配表项
- 发射仲裁：选择最老的就绪分支指令
- 相关性检查：检查分支条件操作数
- 旁路逻辑：支持分支条件数据旁路
