# ct_idu_id_decd 模块详细方案文档

## 1. 模块概述

### 1.1 基本信息

| 属性 | 值 |
|------|-----|
| 模块名称 | ct_idu_id_decd |
| 文件路径 | C910_RTL_FACTORY/gen_rtl _rvv1.0/idu/rtl/ct_idu_id_decd.v |
| 功能描述 | ID阶段指令解码器 - 主解码模块 |

### 1.2 功能描述

ct_idu_id_decd 是指令解码单元(IDU)的ID阶段主解码模块，负责：
- 对32位和16位指令进行解码
- 解析指令的操作数寄存器（源寄存器、目标寄存器）
- 识别指令类型（整数、浮点、向量）
- 处理立即数提取
- 生成RVV(RISC-V Vector)相关控制信号
- 处理特殊指令如fence、atomic操作

### 1.3 设计特点

- 支持RVV 1.0版本扩展
- 支持多种指令格式：R型、I型、S型、B型、U型、J型
- 支持压缩指令(16位)解码
- 支持向量寄存器和浮点寄存器解码
- 支持立即数扩展和格式化

## 3. 解码逻辑说明

### 3.1 指令格式识别

| 指令类型 | 编码格式 | 说明 |
|----------|----------|------|
| 16位指令 | inst[1:0] != 2'b11 | 压缩指令格式 |
| 32位指令 | inst[1:0] == 2'b11 | 标准RISC-V格式 |

### 3.2 立即数解码

模块支持多种立即数格式的提取：
- **I型立即数**: 12位有符号立即数
- **S型立即数**: 12位有符号立即数(用于存储)
- **B型立即数**: 13位有符号立即数(用于分支)
- **U型立即数**: 20位立即数(用于lui/auipc)
- **J型立即数**: 21位有符号立即数(用于跳转)

### 3.3 寄存器解码

| 寄存器类型 | 源/目的 | 有效信号 |
|------------|---------|----------|
| 整数寄存器(x0-x31) | 源/目的 | x_src*_vld / x_dst_vld |
| 浮点寄存器(f0-f31) | 源/目的 | x_srcf*_vld / x_dstf_vld |
| 向量寄存器(v0-v31) | 源/目的 | x_srcv*_vld / x_dstv_vld |
| 向量掩码寄存器(v0) | 源 | x_srcvm_vld |
