# RTU (Retire Unit) 提交单元

## 1. 模块功能说明

RTU (Retire Unit) 是 OpenC910 处理器的提交单元，负责指令的顺序提交、异常处理和状态恢复。RTU 是处理器的关键组件，确保精确异常和顺序语义。

### 主要功能

- **指令提交**: 按程序顺序提交已完成的指令
- **异常处理**: 检测和处理异常，维护精确异常
- **状态恢复**: 在分支误预测或异常时恢复处理器状态
- **中断处理**: 响应外部中断和软件中断
- **ROB 管理**: 管理重排序缓冲区 (Reorder Buffer)
- **寄存器释放**: 提交后释放物理寄存器

### 主要特性

| 特性 | 描述 |
|------|------|
| ROB 条目数 | 128 条目 |
| 提交宽度 | 每周期最多提交 3 条指令 |
| 支持异常类型 | 同步异常、异步异常、中断 |
| 状态恢复 | 支持 Checkpoint 恢复 |
| 调试支持 | 支持硬件断点和单步调试 |
