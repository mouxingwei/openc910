# IU (Integer Unit) 整数单元

## 1. 模块功能说明

IU (Integer Unit) 是 OpenC910 处理器的整数执行单元，负责执行整数运算、分支处理和除法运算。IU 支持多发射和乱序执行。

### 主要功能

- **整数运算**: 执行加、减、逻辑运算、移位等整数操作
- **分支处理**: 执行分支指令，计算分支目标，处理分支误预测
- **乘法运算**: 执行整数乘法，支持 65×65 位乘法
- **除法运算**: 执行整数除法和取余，采用基数16 SRT 算法
- **特殊指令**: 执行 CSR 访问、系统调用等特殊指令

### 主要特性

| 特性 | 描述 |
|------|------|
| ALU 数量 | 2 个 (Pipe0, Pipe1) |
| ALU 延迟 | 1 周期 (大部分运算) |
| 乘法器延迟 | 3-4 周期 |
| 除法器延迟 | 可变 (SRT 算法) |
| 分支单元 | 1 个 (Pipe2) |
| 支持指令集 | RV64IMAC + 扩展 |
