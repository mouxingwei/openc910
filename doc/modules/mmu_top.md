# MMU (Memory Management Unit) 内存管理单元

## 1. 模块功能说明

MMU (Memory Management Unit) 是 OpenC910 处理器的内存管理单元，负责虚拟地址到物理地址的翻译，支持虚拟内存和内存保护。

### 主要功能

- **地址翻译**: 将虚拟地址翻译为物理地址
- **TLB 管理**: 管理指令 TLB (ITLB)、数据 TLB (DTLB) 和联合 TLB (JTLB)
- **页表遍历**: 执行硬件页表遍历 (Page Table Walk)
- **内存保护**: 实现特权级和内存访问权限检查
- **地址空间**: 支持 Sv39 虚拟地址模式

### 主要特性

| 特性 | 描述 |
|------|------|
| 虚拟地址位数 | 39 位 (Sv39) |
| 物理地址位数 | 40 位 |
| 页大小 | 4KB, 2MB, 1GB |
| ITLB 条目数 | 32 条目 + 4 大页条目 |
| DTLB 条目数 | 32 条目 + 4 大页条目 |
| JTLB 条目数 | 512 条目, 4路组相联 |
| PTW 深度 | 3 级页表遍历 |

│  └─────────────────────────────────────────────────────────────┘   │
│                              │                                      │
│                    ┌─────────┴─────────┐                           │
│                    │                   │                           │
│                    ▼                   ▼                           │
│              ┌──────────┐        ┌──────────┐                      │
│              │ TLB Hit  │        │ TLB Miss │                      │
│              └────┬─────┘        └────┬─────┘                      │
│                   │                   │                            │
